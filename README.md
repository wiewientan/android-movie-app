# Android Movie App

Android Movie App is a mobile application developed using Android Studio with Java programming language.
The application allows users to manage movie data and view movie information.

## Features
- Add new movie with details:
  - Movie title
  - Genre
  - Release date
  - Producer
- Display movie list after data is added
- Search movies based on release date
- Update existing movie data
- Delete movie data

## Technologies Used
- Android Studio
- Java
- SQL Database
- PHP (Backend API)

## Purpose
This project was created as an academic assignment for the Mobile Application Development course.
